## 开发手册
### 目录简介
--assets
    |--images  // 存放图片资源
    |--css  // 存放css文件
    |--font  // 存放字体图标资源
--src
    |--pages //存放页面文件
    |--js   // 存放js文件
--lib      // 存放第三方插件 ui库
    |--jq.js  // jq.js文件
    |--bootstrap // bootstrap ui文件
### 开发规范
+ 样式命名 
    + 类名使用小写，以 - 分隔,加以前缀 ynw  例如 ynw-home

+ 响应式
    + css 单位为px,可配合插件转为vw 单位
    + 页面布局使用flex布局
